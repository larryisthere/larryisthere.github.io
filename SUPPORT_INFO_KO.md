# Dawnary - 지원 정보

## 앱 개요

Dawnary는 개인 일기 기록에 중점을 둔 모바일 애플리케이션입니다. 우리는 귀하에게 미니멀하고 치유적인 글쓰기 공간을 제공하여 인생의 모든 순간을 쉽게 기록할 수 있도록 최선을 다하고 있습니다.

## 자주 묻는 질문

### Dawnary 사용 방법은?

1. **기록 시작**: 앱을 연 후 "+" 버튼을 탭하여 새로운 일기 항목 생성
2. **내용 편집**: 리치 텍스트 편집을 지원하며, 이미지를 삽입하고 텍스트 형식을 설정할 수 있습니다
3. **자동 저장**: 앱은 백그라운드로 전환하거나 닫을 때 자동으로 저장됩니다
4. **기록 보기**: 메인 페이지에서 날짜별로 모든 일기 항목을 볼 수 있습니다

### 데이터는 어디에 저장되나요?

- **로컬 저장**: 모든 일기 내용은 기본적으로 기기에 로컬로 저장됩니다
- **iCloud 백업(선택사항)**: 설정에서 iCloud 백업을 활성화하여 데이터를 다른 기기에 동기화할 수 있습니다

### iCloud 백업을 활성화하는 방법은?

1. 앱 설정 열기
2. "iCloud 백업" 옵션 찾기
3. 백업 켜기
4. 첫 동기화에는 시간이 걸릴 수 있으니 잠시 기다려 주세요

### 일기 항목을 삭제하는 방법은?

일기 목록에서 일기 항목을 왼쪽으로 스와이프하고 "삭제" 버튼을 탭하여 삭제합니다.

### 이미지를 삽입하는 방법은?

1. 편집 페이지에서 키보드 도구 모음의 이미지 아이콘 탭
2. "사진 라이브러리에서 선택" 또는 "사진 촬영" 선택
3. 이미지가 현재 커서 위치에 자동으로 삽입됩니다

### 위치 정보는 어떻게 사용되나요?

- 위치 기록은 선택사항입니다
- 활성화하면 앱은 일기 항목을 생성한 도시 이름을 기록합니다(도시 수준만)
- 위치 정보는 일기 항목에 도시 이름을 표시하여 컨텍스트를 향상시키는 목적으로만 사용됩니다
- 위치 정보는 기기에 로컬로만 저장되며 서버에 업로드되지 않습니다
- 설정에서 언제든지 위치 기록을 비활성화할 수 있습니다

### 앱에 인터넷 연결이 필요하나요?

아니요. Dawnary의 핵심 기능은 완전히 오프라인이며 인터넷 연결 없이 작동합니다. 데이터 동기화를 위해 iCloud 백업이 활성화된 경우에만 네트워크 연결이 필요합니다.

### 일기 항목을 내보내거나 공유하는 방법은?

1. 일기 편집 페이지에서 오른쪽 상단의 메뉴 버튼 탭
2. "공유 이미지 생성" 선택
3. 앱이 일기 내용을 이미지 형식으로 변환합니다
4. 사진 라이브러리에 저장하거나 직접 공유할 수 있습니다
