# Dawnary - サポート情報

## アプリ概要

Dawnaryは、個人の日記記録に焦点を当てたモバイルアプリケーションです。私たちは、シンプルで癒しのライティングスペースを提供し、あなたが人生のあらゆる瞬間を簡単に記録できるようにすることを目指しています。

## よくある質問

### Dawnaryの使い方は？

1. **記録を開始**：アプリを開いた後、「+」ボタンをタップして新しい日記エントリを作成
2. **コンテンツを編集**：リッチテキスト編集をサポートし、画像を挿入したり、テキストをフォーマットしたりできます
3. **自動保存**：アプリはバックグラウンドに切り替えたり閉じたりすると自動的に保存されます
4. **履歴を表示**：メインページで日付別にすべての日記エントリを表示できます

### データはどこに保存されますか？

- **ローカル保存**：すべての日記内容は、デフォルトでデバイスにローカルに保存されます
- **iCloudバックアップ（オプション）**：設定でiCloudバックアップを有効にして、データを他のデバイスに同期できます

### iCloudバックアップを有効にするには？

1. アプリ設定を開く
2. 「iCloudバックアップ」オプションを見つける
3. バックアップをオンにする
4. 初回同期には時間がかかる場合がありますので、しばらくお待ちください

### 日記エントリを削除するには？

日記リストで、日記エントリを左にスワイプし、「削除」ボタンをタップして削除します。

### 画像を挿入するには？

1. 編集ページで、キーボードツールバーの画像アイコンをタップ
2. 「フォトライブラリから選択」または「写真を撮る」を選択
3. 画像が現在のカーソル位置に自動的に挿入されます

### 位置情報はどのように使用されますか？

- 位置記録はオプションです
- 有効にした場合、アプリは日記エントリを作成した都市名を記録します（都市レベルのみ）
- 位置情報は、日記エントリに都市名を表示してコンテキストを強化するためだけに使用されます
- 位置情報はデバイスにローカルにのみ保存され、サーバーにアップロードされることはありません
- 設定でいつでも位置記録を無効にできます

### アプリにはインターネット接続が必要ですか？

いいえ。Dawnaryのコア機能は完全にオフラインで、インターネット接続なしで動作します。データ同期のためにiCloudバックアップが有効になっている場合のみ、ネットワーク接続が必要です。

### 日記エントリをエクスポートまたは共有するには？

1. 日記編集ページで、右上のメニューボタンをタップ
2. 「共有画像を生成」を選択
3. アプリが日記内容を画像形式に変換します
4. フォトライブラリに保存するか、直接共有できます
