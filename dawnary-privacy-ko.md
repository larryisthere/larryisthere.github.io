## 개요

Dawnary("당사", "우리" 또는 "이 앱")는 개인 일기 기록에 중점을 둔 모바일 애플리케이션입니다. 우리는 귀하의 개인정보를 매우 중요하게 생각하며, 이 개인정보 보호정책은 우리가 정보를 수집, 사용, 저장 및 보호하는 방법을 자세히 설명합니다.

### 데이터 저장 방법

**로컬 저장:**
- 모든 일기 내용은 기본적으로 기기에 로컬로 저장됩니다
- 데이터를 제3자 서버에 업로드하지 않습니다
- 일기 내용에 액세스, 읽기 또는 전송하지 않습니다
- 데이터를 제3자와 공유하지 않습니다

**iCloud 백업(선택사항):**
- iCloud 백업 기능을 활성화하면 일기 데이터가 Apple의 iCloud 서비스를 통해 기기 간에 동기화됩니다
- iCloud 백업은 Apple ID를 사용하며, 데이터는 Apple의 iCloud 서버에 저장됩니다
- **데이터 저장 위치:** iCloud 데이터는 귀하의 국가/지역 외부에 있는 서버에 저장될 수 있으며, 국경을 넘는 데이터 전송이 포함될 수 있습니다
- **국경을 넘는 전송에 대한 알림:** iCloud 백업 기능을 사용하는 경우 데이터가 국가/지역 외부에 있는 Apple의 서버로 전송될 수 있습니다. 이 데이터 전송은 Apple Inc.에서 관리하며, Apple의 개인정보 보호정책 및 이용 약관의 적용을 받습니다. 설정에서 언제든지 iCloud 백업을 비활성화하여 국경을 넘는 데이터 전송을 중지할 수 있습니다
- iCloud 데이터는 Apple의 개인정보 보호정책 및 이용 약관에 의해 보호됩니다

### 데이터 사용 목적

로컬에 저장된 데이터는 다음 목적으로만 사용합니다:
- 일기 기록 및 편집 기능 제공
- 일기 목록 표시
- 애플리케이션 환경 설정 저장
- 위치 정보 기록(활성화된 경우): 일기 항목에 도시 이름을 표시하여 컨텍스트 향상
- 공유 이미지 생성: 일기 내용을 이미지 형식으로 변환하여 다운로드 또는 공유

## 권한 사용

이 앱은 다음 권한이 필요할 수 있으며, 모든 권한은 선택사항입니다:

**사진 라이브러리 권한:**
- 목적: 사진 라이브러리에서 이미지를 선택하여 일기 항목에 삽입하거나 공유 이미지를 사진 라이브러리에 저장할 수 있도록 허용
- 사용 방법: 이미지를 적극적으로 선택하거나 저장할 때만 사용
- 데이터 수집: 사진 라이브러리 내용을 수집하거나 업로드하지 않습니다

**카메라 권한:**
- 목적: 사진을 촬영하여 일기 항목에 삽입할 수 있도록 허용
- 사용 방법: 카메라 기능을 적극적으로 사용할 때만 사용
- 데이터 수집: 촬영한 사진을 수집하거나 업로드하지 않습니다

**위치 권한(민감한 개인정보):**
- 목적: 일기 항목을 생성한 도시 이름을 기록(도시 수준만)하여 일기 항목에 도시 정보를 표시하고 컨텍스트를 향상시킴
- 필요성 설명: 위치 정보는 일기의 컨텍스트 기록을 향상시키는 데 도움이 되지만, 이 기능은 완전히 선택사항이며 앱의 핵심 기능에 영향을 주지 않습니다
- 사용 방법: 명시적으로 동의하고 위치 기록 기능을 활성화한 경우에만 사용
- 데이터 수집: "서울시"와 같은 도시 이름만 기록하며, 정확한 GPS 좌표나 기타 위치 데이터는 기록하지 않습니다
- 데이터 저장: 위치 정보는 기기에 로컬로만 저장되며 서버에 업로드되지 않습니다
- 제어 방법: 설정에서 언제든지 위치 기록 기능을 비활성화할 수 있습니다. 비활성화하면 위치 정보를 더 이상 수집하지 않습니다

모든 권한은 시스템 설정에서 언제든지 취소할 수 있습니다.
