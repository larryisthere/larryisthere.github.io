# Dawnary - 支持信息

## 应用简介

Dawnary 是一款专注于个人日记记录的移动应用程序。我们致力于为您提供一个极简、治愈的写作空间，让您能够轻松记录生活中的每一个瞬间。

## 常见问题

### 如何使用 Dawnary？

1. **开始记录**：打开应用后，点击"+"按钮创建新的日记
2. **编辑内容**：支持富文本编辑，可以插入图片、设置文字格式
3. **自动保存**：应用会在您切换后台或关闭时自动保存
4. **查看历史**：在主页面可以按日期查看所有日记记录

### 数据存储在哪里？

- **本地存储**：所有日记内容默认存储在您的设备本地
- **iCloud 备份（可选）**：您可以在设置中启用 iCloud 备份，将数据同步到您的其他设备

### 如何启用 iCloud 备份？

1. 打开应用设置
2. 找到"iCloud 备份"选项
3. 开启备份功能
4. 首次同步可能需要一些时间，请耐心等待

### 如何删除日记？

在日记列表中，向左滑动日记条目，点击"删除"按钮即可删除该日记。

### 如何插入图片？

1. 在编辑页面，点击键盘工具栏上的图片图标
2. 选择"从相册选择"或"拍照"
3. 图片会自动插入到当前光标位置

### 位置信息是如何使用的？

- 位置记录功能是可选的
- 如果启用，应用会记录您创建日记时所在的城市名称（仅城市级别）
- 位置信息仅用于在日记中显示城市名称，增强日记的情境感
- 位置信息仅存储在您的设备本地，不会上传到任何服务器
- 您可以随时在设置中关闭位置记录功能

### 应用需要网络连接吗？

不需要。Dawnary 的核心功能完全离线，无需网络连接即可使用。只有在启用 iCloud 备份时，才需要网络连接进行数据同步。

### 如何导出或分享日记？

1. 在日记编辑页面，点击右上角的菜单按钮
2. 选择"生成分享图片"
3. 应用会将日记内容转换为图片格式
4. 您可以保存到相册或直接分享
